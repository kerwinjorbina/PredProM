Predictive Monitoring

Hello there Process Mining Enthusiast!

You are now in the PredProM (Predictive Process Mining) Project. This is an instructional material for those who would like to begin their journey in the predictive process monitoring realm in Process Mining. This explains how to encode the historical business process log in preparation for further usages such as predictive process monitoring. Usually, the logs are formatted as XES or MXML (BPM Standards for logs) files which are just XML files. For one to perform machine learning algorithms such as classification and regression, one must preprocess these logs into data frames (or in CSV format) so we can feed it directly to the learning techniques.

The encoders implemented in this project is based on the theory presented in a paper published during the Business Process Management (BPM) 2015 Conference by Anna Leontjeva et al. The paper can be seen at this link.

This is a hobby project by me (Kerwin). This uses Angular 2 for the front-end application. The implementation of the encoders can be found at the Nirdizati-Encoders.
