Boolean Encoding

To perform this encoding method, we take all the possible events in a given trace and convert them to the columns of our dataset. We assign a true or false value if the event occurs on a given trace. Let us see the example below:

We can see that there are 5 unique events name A, B, C, D and E. These will now be our columns in the encoded data. In addition, we used the remaining time as the metric of interest for the prediction.
